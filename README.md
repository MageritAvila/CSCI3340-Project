# CSCI3340-Project

Project Description: This app is a to-do bulletin board for coworkers. Users will utilize this application to post tasks needing to be done. Faculty can also decide to delete or update a task if there is need for change. These tasks will be categorized into specific projects. Users are able to pick up multiple tasks within a project; however, they cannot pick up task from another project until all tasks in their current project have been completed. Task must have a due date, title, description, and optionally an icon to be published. The optional icon will determine whether the task is urgent or not.  Tasks will have a different color based on their status. Grey means unclaimed, yellow means in progress, green means completed, and red means overdue. Users must create an account before claiming a task to ensure who will be preforming the task. This project will follow CRUD allowing users to create, read, update and delete.
